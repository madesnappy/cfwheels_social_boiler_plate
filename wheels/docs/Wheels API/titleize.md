# titleize()

## Description
Capitalizes all words in the text to create a nicer looking title.

## Function Syntax
	titleize( word )

	</tbody>

## Examples
	
		#titleize("Wheels is a framework for ColdFusion")#
		-> Wheels Is A Framework For ColdFusion
